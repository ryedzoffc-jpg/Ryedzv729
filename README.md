A modern chat application with WhatsApp-like features plus unique additions like floating chat heads and admin observer mode.

## ✨ Features

- 🔐 **Phone & Google Authentication** - Secure login with OTP and Google Sign-In
- 💬 **Real-time Chat** - Instant messaging with Firebase Firestore
- 🔒 **End-to-End Encryption** - AES encryption for all messages
- 🆔 **Unique 5-Digit ID** - Easy contact sharing system
- 💭 **Floating Chat Heads** - Chat while using other apps (like Messenger)
- 👁️ **Admin Observer Mode** - Monitor all conversations as admin
- 🟢 **Online/Offline Status** - Real-time user presence
- ✓✓ **Message Status** - Sent, Delivered, Read receipts
- 👤 **Profile Management** - Customizable user profiles
- 🎨 **Modern UI/UX** - Glassmorphism design with smooth animations
- 📱 **PWA Support** - Install on mobile devices
- 🔔 **Push Notifications** - Stay updated with new messages

## 🛠️ Tech Stack

- **Frontend:** React 18 + Vite
- **Styling:** Tailwind CSS + Framer Motion
- **Backend:** Firebase (Auth, Firestore, Storage)
- **Security:** AES Encryption (CryptoJS)
- **State:** Context API + Custom Hooks
- **Routing:** React Router v6
- **Icons:** React Icons
- **Notifications:** React Hot Toast
